# Quick Start Guide

This guide will help you get started with the Cancer Treatment Optimization project quickly.

## Prerequisites

Before you begin, ensure you have:
- Python 3.8+ installed
- R 4.0+ installed (optional, for R scripts)
- Git installed
- Basic understanding of Python programming

## Installation in 5 Minutes

### Step 1: Clone the Repository

```bash
git clone https://github.com/yourusername/cancer-treatment-optimization.git
cd cancer-treatment-optimization
```

### Step 2: Set Up Python Environment

#### Option A: Using pip and venv

```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

#### Option B: Using Conda

```bash
# Create conda environment
conda env create -f environment.yml

# Activate environment
conda activate cancer-treatment-opt
```

### Step 3: Verify Installation

```bash
python -c "import numpy, pandas, sklearn; print('All packages installed successfully!')"
```

## Running Your First Analysis

### Option 1: Run the Python Script (Quickest)

```bash
python scripts/cancer_treatment_optimization.py
```

This will:
1. Generate synthetic cancer treatment data
2. Train a machine learning model to predict treatment response
3. Optimize drug dosing schedules
4. Generate visualizations

### Option 2: Use the Jupyter Notebook (Interactive)

```bash
# Start Jupyter Notebook
jupyter notebook notebooks/cancer_treatment_modeling.ipynb
```

Then run all cells to execute the complete analysis pipeline.

### Option 3: Run R Statistical Analysis

```bash
# Install R packages first (one-time setup)
Rscript -e "install.packages(c('dplyr', 'ggplot2', 'survival', 'survminer', 'DESeq2', 'pheatmap', 'corrplot', 'RColorBrewer'))"

# Run the analysis
Rscript scripts/statistical_analysis.R
```

## Understanding the Output

After running any of the above methods, you'll find:

- **Generated plots**: PNG files showing analysis results
- **Model metrics**: Performance statistics printed to console
- **Optimization results**: Drug dosing recommendations

## Common Issues

### Issue: ImportError for sklearn

**Solution**: Make sure you've activated your virtual environment and installed requirements:
```bash
pip install scikit-learn
```

### Issue: Jupyter notebook doesn't start

**Solution**: Install jupyter:
```bash
pip install jupyter
```

### Issue: R packages fail to install

**Solution**: Use BiocManager for Bioconductor packages:
```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install(c("DESeq2"))
```
