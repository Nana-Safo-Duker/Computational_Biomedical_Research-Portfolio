# Image-based cell phenotyping with deep learning

## Overview
This repository contains a scientific blog adaptation, reproducible analysis scripts, and notebook workflows for the topic: **Image-based cell phenotyping with deep learning**.

## Repository Structure
- Guidelines_Research_Paper_Review.txt: review-to-blog writing guide
- log_post.md: scientific blog post adapted from the guideline
- .ipynb, .py, .R: detailed computational workflows
- .gitattributes, .gitignore, LICENSE, 
equirements.txt: repository setup files

## Reproducibility
1. Create a Python environment.
2. Install dependencies from 
equirements.txt.
3. Run the Python and R scripts.
4. Open the notebook for interactive analysis and figures.

## Notes
This project is structured for publication-style transparency and educational reuse.
